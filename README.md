# Number-guessing-game
A simple, interactive command-line number guessing game built with python

 🎯 Python Number Guessing Game

An interactive command-line game where the computer selects a random secret number between 1 and 100, and the player tries to guess it using helpful hints. This is my second Python project, focusing on implementing conditional game logic and managing loop control states.

---

 🚀 Features

* **Random Number Generation:** Automatically picks a new secret number between 1 and 100 every time a new game starts.
* **Dynamic Feedback:** Gives the player instant feedback ("Too high!" or "Too low!") to guide their next guess.
* **Win Condition:** Detects when the player successfully guesses the correct number, displays a congratulatory message, and gracefully exits.

---

 🛠️ Prerequisites

To run this game, you only need **Python 3.x** installed on your computer. It relies entirely on Python's built-in `random` module, so there are no external dependencies to install.
